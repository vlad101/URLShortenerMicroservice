API Basejump: URL Shortener

Instructions:
  • User can pass a URL as a parameter and user will receive a shortened URL in the JSON response.
  • When user visits that shortened URL, it will redirect user to the original link.

Example usage:

http://{{main.host}}:{{main.port}}/api/shortener/newUrl/https://www.google.com
http://{{main.host}}:{{main.port}}/api/shortener/newUrl/http://freecodecamp.com/news

Example output:

{ "original_url": "http://freecodecamp.com/news", "short_url": "http://{{main.host}}:{{main.port}}/4" }
